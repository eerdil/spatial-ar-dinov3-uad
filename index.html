Spatial Autoregressive Modeling of DINOv3 Embeddings for Unsupervised Anomaly Detection

MICCAI 2025 submission (anonymized) • BMAD benchmark (BraTS2021, BTCV+LiTs, RESC)
Authors: Anonymized Authors
Paper (PDF) Code Poster Slides

Quick facts

Core idea: 2D masked CNN autoregressive model on DINOv3 patch embeddings
Inference: single forward pass (no kNN / memory bank)
Efficiency: ~20 ms/image on RESC (A6000)
Benchmark: BMAD (BraTS2021, BTCV+LiTs, RESC)

Abstract

DINO models provide rich patch-level representations that have recently enabled strong performance in unsupervised anomaly detection (UAD). Most existing methods model patch embeddings independently, ignoring spatial relationships and often requiring large memory banks and costly comparisons. We propose a simple and efficient framework that explicitly models spatial dependencies between DINOv3 embeddings using a 2D autoregressive (AR) CNN with masked (and optionally dilated) convolutions. At test time, anomaly detection reduces to a single forward pass, enabling fast and memory-efficient inference. We evaluate on BMAD (BraTS2021, BTCV+LiTs, RESC) and show competitive AUROC/AUPR while substantially reducing runtime and memory.

Method overview

We extract a patch grid of embeddings F = Φ(x) ∈ ℝ^{Hₚ×Wₚ×D} and model the joint distribution with a raster-scan AR factorization. A masked CNN predicts conditional means, and per-patch negative log-likelihood becomes the anomaly score. Dilated masked convolutions expand receptive field under the AR constraint to reduce “local interpolation” effects in anatomically structured data.
